# Bash Setup Core

A blazing-fast, zero-dependency, pure Bash package manager and environment sync tool. 

## Features
- **Tool Management:** Dynamically installs pre-compiled binaries (fd, ripgrep, fzf, etc.) directly from GitHub.
- **System Sync:** Distro-agnostic system package manager (`apt`, `pacman`, `dnf`, `zypper`).
- **Runtime Sync:** Automatically provisions languages (Node, Python, Go, Rust) via `mise`.
- **Self-Updating:** Includes a built-in `tool update` mechanism to stay current.
